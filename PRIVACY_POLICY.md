# プライバシーポリシー / Privacy Policy

最終更新日: 2026年1月20日

## ShieldAI - 機密情報フィルタ

### データの収集について

ShieldAIは、ユーザーのプライバシーを最優先に設計されています。

**収集しないデータ:**
- 入力されたテキスト
- 検出された機密情報
- 個人を特定できる情報
- 使用状況や統計データ

### データの処理

- すべての処理はユーザーの端末内（ブラウザ内）で完結します
- 外部サーバーへのデータ送信は一切行いません
- インターネット接続がなくても動作します

### ローカルストレージ

以下のデータのみ、ブラウザのローカルストレージに保存されます：
- カスタム辞書（ユーザーが登録したNGワード）

このデータは端末内にのみ保存され、外部に送信されることはありません。

### 第三者への提供

ユーザーのデータを第三者に提供、販売、共有することはありません。

### お問い合わせ

ご質問やご懸念がある場合は、GitHubリポジトリのIssueにてご連絡ください。
https://github.com/kmura-09/shield-ai/issues

# Privacy Policy (English)

Last updated: January 20, 2026

## ShieldAI - Sensitive Information Filter

### Data Collection

ShieldAI is designed with user privacy as the top priority.

**Data NOT collected:**
- Text entered by users
- Detected sensitive information
- Personally identifiable information
- Usage statistics or analytics

### Data Processing

- All processing occurs entirely within the user's device (browser)
- No data is ever sent to external servers
- Works without an internet connection

### Local Storage

Only the following data is stored in the browser's local storage:
- Custom dictionary (user-registered NG words)

This data is stored only on the device and is never transmitted externally.

### Third-Party Sharing

We do not provide, sell, or share user data with any third parties.

### Contact

If you have questions or concerns, please contact us via GitHub Issues:
https://github.com/kmura-09/shield-ai/issues
