# Chrome Web Store 掲載情報

## 拡張機能名
ShieldAI - 機密情報フィルタ

## 短い説明（132文字以内）
ChatGPT/Claude/Geminiに送信する前に機密情報を検出。メール、電話番号、会社名、個人名などを自動マスク。完全ローカル処理でデータ外部送信なし。

## 詳細説明

AIチャットに機密情報をうっかり送信していませんか？

ShieldAIは、ChatGPT・Claude・Geminiなどの生成AIに送信する前に、テキスト内の機密情報を検出してマスクするツールです。

【主な機能】
- メールアドレス、電話番号、住所の自動検出
- 金額（¥表記、○○万円）の検出
- 会社名パターン（株式会社○○、○○Inc.など）の検出
- 敬称付き名前（○○様、○○さん）の検出
- APIキー、クレジットカード番号の検出
- カスタム辞書で取引先名やプロジェクト名を登録可能

【特徴】
- 完全ローカル処理：データは一切外部に送信されません
- 日本語特化：日本の電話番号、住所形式、敬称に対応
- リアルタイム警告：ChatGPT/Claude/Gemini入力時に警告表示
- 辞書のエクスポート/インポート対応

【使い方】
1. 拡張機能アイコンをクリック
2. テキストを貼り付けて「チェック」
3. マスク済みテキストをコピーしてAIに送信

【対象ユーザー】
- AIを業務で使いたいが情報漏洩が心配な方
- 顧客情報を扱うビジネスパーソン
- 機密性の高い文書をAIで処理したい方

【注意事項】
- 本ツールは正規表現ベースの簡易チェックです
- 100%の検出を保証するものではありません
- 重要な機密情報は送信前に必ず目視確認をお願いします

【プライバシー】
- 入力されたテキストは端末内でのみ処理されます
- 外部サーバーへのデータ送信は一切行いません
- 辞書データはブラウザのローカルストレージに保存されます
