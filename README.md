# ShieldAI - 機密情報フィルタ

ChatGPT/Claude/Gemini等の生成AIに送信する前に、テキスト内の機密情報を検出してマスクするChrome拡張機能です。

## 特徴

- **完全ローカル処理** - データは一切外部に送信されません
- **日本語特化** - 日本の電話番号、住所、敬称付き名前等に対応
- **カスタム辞書** - 取引先名やプロジェクト名を登録可能
- **リアルタイム警告** - ChatGPT/Claude/Gemini入力時に警告表示

## スクリーンショット

```
入力: 田中様（03-1234-5678）にメール info@example.com で連絡。金額は¥500,000です。
出力: [個人名]（[電話番号]）にメール [メールアドレス] で連絡。金額は[金額]です。
```

## インストール

### Chrome Web Store（準備中）
近日公開予定

### 開発版
1. このリポジトリをクローン
2. Chrome で `chrome://extensions/` を開く
3. 「デベロッパーモード」を有効化
4. 「パッケージ化されていない拡張機能を読み込む」で `extension` フォルダを選択

## 使い方

1. 拡張機能アイコンをクリック
2. テキストを貼り付けて「チェック」
3. 検出された機密情報を確認
4. マスク済みテキストをコピーしてAIに送信

## 検出対象

| カテゴリ | 例 |
|---------|-----|
| メールアドレス | example@test.com |
| 電話番号 | 03-1234-5678、090-1234-5678 |
| 住所 | 東京都渋谷区..., 横浜市中区... |
| 郵便番号 | 〒123-4567 |
| 金額 | ¥100,000、500万円 |
| 会社名 | 株式会社ABC、有限会社XYZ |
| 敬称付き名前 | 田中様、山田さん、佐藤氏 |
| マイナンバー | 1234-5678-9012 |
| 銀行口座 | 口座番号: 1234567 |
| クレジットカード | 4111-1111-1111-1111 |
| APIキー | sk-xxx...、AKIA... |
| IPアドレス | 192.168.1.1 |

## カスタム辞書

辞書タブから、マスク対象の単語を登録できます：
- 取引先名（トヨタ自動車、A社 等）
- プロジェクト名（案件X 等）
- 個人名（山田太郎 等）
- その他NGワード

辞書はエクスポート/インポート可能です。

## 開発

```bash
# テスト実行
cd extension
node tests/detector.test.js
```

## 注意事項

- 本ツールは正規表現ベースの簡易チェックです
- 100%の検出を保証するものではありません
- 重要な機密情報は送信前に必ず目視確認をお願いします

## プライバシー

- 入力されたテキストは端末内でのみ処理されます
- 外部サーバーへのデータ送信は一切行いません
- 詳細は [PRIVACY_POLICY.md](PRIVACY_POLICY.md) を参照
